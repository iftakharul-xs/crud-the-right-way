Project Requirement
====================


An offer will have the following properties:

* Title
* Price
* Categories
* Locations
* Image (optional)
* Description

#### There will be two types of users:

### Admin

### Seller

Seller can:

* Create an "Offer"
* Edit an offer
* Delete an offer
* View a list of their offers


Admin can:

* View a list of all offers
* Edit an offer
* Delete an offer
